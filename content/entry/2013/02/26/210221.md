C++ で新しい演算子“-->” が採用されたそうだが、これは C# ですでに利用可能である。この演算子は、左辺の変数を右辺値まで 1 ずつ減算する。
```cs
@{
    int x = 10;
}

&lt;!DOCTYPE html>

&lt;html lang="ja">
    &lt;head>
        &lt;meta charset="utf-8" />
        &lt;title>マイ サイトのタイトル&lt;/title>
    &lt;/head>
    &lt;body>
        @do {
            &lt;p>@x&lt;/p>
	}
	while(x --> 0);
    &lt;/body>
&lt;/html>

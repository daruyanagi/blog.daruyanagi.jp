### Desktop IE

esktop IE 11 にはほとんど変更はないみたい（ただし、タブ関連の機能に若干の差異がある。参照: IE10 と IE11 の［インターネット オプション］を比較 - だるろぐ）。本題には関係ないけれど、スクリーンショットでは Office の IE アドインが拡張保護モードによってブロックされているのがわかる。ちゃんと拡張保護モードが動いているのだな、と安心するとともに、自社アプリなんだから対応しておこうよ……と思わないでもない。

### Immersive IE Immersive IE 11（要するにデスクトップじゃない、メトロな IE 11）には変更が多い。

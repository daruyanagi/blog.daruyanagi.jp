Windows のあのフォルダーをサクッと開きたい！　覚えておくとちょっと便利なシェルコマンド

公開日： 2015/09/15

ょっと覚えておくと便利なシェルコマンドの一覧。たとえば［Windows］＋［R］キーで「ファイルを指名して実行」を開いて、著ちょいと入力してエンターを押すと目当てのフォルダーへサクッとアクセスできる。OS起動時に実行開始するアプリを登録する“shell:Startup”辺りは割と使うかな？

Surface RT：（本体側の）［Windows］キーの機能

公開日：2013/05/09

Surface RT のベゼル下部に設けられている Windows ロゴキーを押すと、現在表示中の画面とスタート画面を切り替えることができる。押すとブルッと振動するフィードバック機能がちょっとイケてる1。けれど、このボタン、ほかにもいろんなことに使えるんだよ。ただし、キーボードの［Windows］キーでは使えないので注意2。

Windows 8 RP のスクリーンショット機能

公開日：2012/06/06

Windows 8 は［Windows］＋［PrintScreen］キーでデスクトップのスクリーンショットを“ピクチャー”フォルダに保存できる - だるろぐWindows 8 CP ではスクリーンショットが「ピクチャ」フォルダに保存されていたけど、Windows 8 RP では「ピクチャ\スクリーンショット」フォルダに保存されるようになったみたい。というわけで、08式机上撮影機 v1.4.0 - だるろぐ を少しだけ手直ししようかと思う。

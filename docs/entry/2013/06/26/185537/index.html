「Mozilla Firefox 22」における細かい変更

公開日：2013/06/26

### インスペクタでフォントのプレビューが可能に

Mozilla Firefox 21」の場合。

Mozilla Firefox 22」の場合。ページでつかわれているフォントをすべてプレビューすることもできるみたい。

### プレーンテキストの折り返し

Mozilla Firefox 21」の場合。

Mozilla Firefox 22」の場合。右端で折り返されるので、行が長くても横スクロールが不要。

Mozilla Firefox 32 Beta の細かいデザイン変更

公開日：2014/07/28


### ページのコンテキストメニュー テキストが選択されていない場合は、戻る・進む・リロード・ブックマークに追加といったコマンドが追加される。
### 接続エラー メッセージは変わっていないので純粋にデザインの変更だけみたい。注意アイコンが三角から丸になったのはなぜなんだろう。個人的には道路標識のイメージで三角のほうが注意を促すという意味ではいいと思うのだけど、ユニバーサルデザインとしてはこちらが適切なのかな。 ちなみにこれまではこんな感じだった。

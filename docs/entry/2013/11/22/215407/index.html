Peek Help Solution Explorer Errors Structure Visualizer Double click to maximize windows Timestamp margin Quick tasks – Edit Present On Ctrl + Click to Peek Definition HTML Copy improvements Recently Closed Documents Match Margin Power Commands context menu cleanup

ClickOnce インストーラーで「ハードディスクへの書き込みエラーが発生しました。ディスクに十分な空きがあるかどうか確認してください」と表示される

公開日：2012/03/09

ClickOnce インストーラーで、ディスクにGB単位で余裕があるにもかかわらず、こんなエラーが表示される場合がある。ウチの場合は、同じDLLが2回インストールされようとしていたのが原因らしい。プロジェクトのプロパティにある［発行］タブで［アプリケーションファイル］ボタンを押し、重複ファイルを除外してやればエラーが解消される。

WebMatrix： URL にドットを含めたい

公開日： 2015/09/13

iki エンジンなんかを作るときなど、「ドット（.）」を URL に含めたい場合は、Web.config に一行、以下のように加えるといいみたい。

<!--?-->xml version="1.0" encoding="UTF-8"?>
<configuration></configuration>
    .webServer>
        runAllManagedModulesForAllRequests="true" />
    .webServer>

##### 1.2.6

Fixed misbehavior when updating attached pictures in files with ID3v2.2 tags. ReplayGain tagging now works with AIFF. Fixed crashing when embedding specific picture files into Ogg Vorbis files. Attached picture manager dialog tweaks.

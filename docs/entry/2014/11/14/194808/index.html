WebMatrix： 型または名前空間 &#39;Linq&#39; は名前空間 &#39;System&#39; に存在しません。アセンブリ参照が不足しています。

公開日： 2014/11/14

br/>

#### 問題 WebMatrix で作った ASP.NET Web Pages のプロジェクトを Visual Studio で開くと、 
>
    型または名前空間 &#39;Linq&#39; は名前空間 &#39;System&#39; に存在しません。アセンブリ参照が不足しています。

というエラーが出てコンパイルできない。

#### 解決策 Web.config で明示的にターゲットフレームワークを指定する。WebMatrix はこれがなくても動くのだけど、Visual Studio の方はちゃんと書いておかないと動かない。 ```xml xml version="1.0"?> .web> targetFramework="4.0" debug="true"/> .web>

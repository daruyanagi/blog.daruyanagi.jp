Facebook 1.3.0.9

公開日：2014/03/03


### Facebook 1.2.0.12 → 1.3.0.9 
>
    New features:

Turn off chat Narrower margins for snap view to show more content Create/join/leave groups Black and white high contrast modes Edit privacy when editing posts Double-tap to zoom photos full res Accessibility enhancements Bugfixes Note: Windows 8.1 is required to run this app.

#### Turn off chat チャットのコンタクトリスト下部に設定ボタンが追加されている。 チャットをオフにした状態。
#### Create/join/leave groups 「あ、なかったんだー」って感じの機能。作成は左ペインで、参加と退会はグループ画面で参加人数の欄をタップ。参加者一覧の左上にある設定ボタンから。これはとても分かりにくいと思った。
#### Edit privacy when editing posts これはないと困る人も多いのじゃないかな。

、これがハンバーガー（チリソースだっけかに卵焼きをトッピング）。1,000円ぐらいだったかな。意外に本格的で、結構おいしかったです。バンズは硬めだけど、肉汁に負けず、食べごたえもある。ひとつ食べただけでお腹いっぱい。ごちそうさまでした。

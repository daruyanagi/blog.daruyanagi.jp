散髪屋を求めて

公開日：2012/03/03

天気が良かったので、散髪屋を求めて散歩に出かけた。すると、10分程度歩いた場所に見つけた。おばちゃんに髪を切ってもらった。4,000円。あいさつをして帰ると、すぐそばにおしゃれな、今風の散髪屋というか美容室があった。でも、わしはああいう、古いほうが落ち着く。逆に、1,000円散髪で済ますのもなんだかなって思うしね。

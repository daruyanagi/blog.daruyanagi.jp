実践 Fiddler

こんな記事（【レビュー】Webアプリのデバッグに最適。HTTPに特化したネットワークキャプチャーツール「Fiddler」 - 窓の杜）読むぐらいなら、これ買った方がいい。3,360円だったら十分ペイすると思う。

この本読みながら1ヶ月ぐらい「Fiddler」使ってみたけど、もはや手放せないツールですな。FiddlerScript バリバリ書くとか言うレベルには全然達していないけれどｗ

わりと WireShark（Wireshark - 窓の杜ライブラリ） とか Microsoft MessageAnalyzer（Microsoft、ネットワーク解析ツール「Microsoft Message Analyzer」を正式公開 - 窓の杜）と一緒くたにしているひとが多い気がするので、独断と偏見で違いを説明してみる。

  • WireShark: パケットをみるやつ（低レベル）。どんなプロトコルでもディセクターさえあればなんでもござれ。
  • Microsoft MessageAnalyzer: トラフィックの傾向を分析するのに便利そう。むろん、詳細も見れるけれど。
  • Fiddler: HTTP トラフィックをみるやつ（高レベル）。Web アプリのリクエスト・レスポンスをみるとか、あとプロキシサーバー的に使ってもイイ。

使い比べたらわかるけど、結構違うと思う。

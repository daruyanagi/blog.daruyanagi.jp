とりあえず AppData 以下の関連フォルダを削除すればいいのね。

USERPROFILE%\AppData\Local\Microsoft\WebMatrix が関係ありそうなので（これ意外にもある？）……

れをを削除して……

度アンインストール。

でもう一回 WebMatrix をインストールして……

uGet Pakage Manager を更新。そして WebMatrix を再起動させると……どうやら今回は成功したみたい！

Formula Pad 1.0.1

公開日： 2013/05/02

キストボックスに TeX 記法で数式を入力すると、それを画像にしてくれます。数式画像はクリップボードへコピーしたり、ファイルとして保存可能。数式テキストを“はてな記法”でコピーすることもできます*1。“Google Chart API”を利用しているので、ネットワーク回線が必須。

得な機能として、

text = text.Replace(@"\land", @"\wedge");
text = text.Replace(@"\lor", @"\vee");
text = text.Replace(@"\lnot", @"\neg");

```という処理が入っています。“Google Chart API”や“はてな”では使えない“\land（論理和）”“\lor（論理積）”“\lnot（否定）”が使えます。
注意 この記事は はてなブログ からのインポートしましたが、まだメンテナンスが終わっていないため、デザインが崩れている可能性があります。読みにくい場合やテキストに欠損がある場合は こちら を参照してください。

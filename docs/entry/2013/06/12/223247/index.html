#### June 2013 updates
Released June 11Update includes these firmware and performance improvements:

Surface Home Button driver update that improves the reliability of the devices waking up immediately on the first button press. Firmware update for Surface Type Cover and Surface Touch Cover that improves functionality for both accessories. Includes the ability to toggle function keys, new shortcuts, and helpful keyboard navigation improvements.

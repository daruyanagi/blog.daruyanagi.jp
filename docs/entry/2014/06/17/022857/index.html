It can run alongside and independently of IE11”とのことなのでそのままぶち込む。対応は Windows 7 SP 1 と Windows 8.1。

りあえず HTML5test - How well does your browser support HTML5? を走らせる。

常版から2ポイント下がって「ふぁっ！？」ってなった。リロードしても同じ。そのうち DP がフリーズしたので、タスクマネージャーで殺してもう一回起動してみたら――

ぜか6ポイントあがった（通常版は4ポイント下がった）。この結果（通常版 372、DP1 378）が多分正しい。そのほかにも割といろいろよくわかんない動作するので、利用の際は注意されたい。なんか重いし。あと“WebGL improvements”って何かなーと思ったので、バージョンを確かめてみた（IE11 の WebGL）。

んのり上がってた（0.93 → 0.94）。

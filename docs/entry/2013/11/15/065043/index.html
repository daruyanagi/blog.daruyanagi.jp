Google Chrome 32 と Web 検索

公開日：2013/11/15

### Google Chrome 31 Google Chrome 31 でオムニバーから Web 検索すると、「普通の」Google 検索の結果ページが現れる。
### Google Chrome 32 同じことを Google Chrome 32 で行うと、
  • Google 検索画面にキーワード入力欄がない
  • オムニバーのアイコンが検索マークに
  • 音声検索機能が統合されている（オムニバー右端のマイクアイコン）
たしかに検索ボックスは2つも要らないし、いい改善なのかもしれない。

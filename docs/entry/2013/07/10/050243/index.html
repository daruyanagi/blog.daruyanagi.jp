公開日：2013/07/10

### Desktop IE Desktop IE 11 にはほとんど変更はないみたい（ただし、タブ関連の機能に若干の差異がある。参照: IE10 と IE11 の［インターネット オプション］を比較 - だるろぐ）。本題には関係ないけれど、スクリーンショットでは Office の IE アドインが拡張保護モードによってブロックされているのがわかる。ちゃんと拡張保護モードが動いているのだな、と安心するとともに、自社アプリなんだから対応しておこうよ……と思わないでもない。

はてなブログ1周年おめでとう！

公開日： 2012/11/07

via はてなブログ1周年！ ありがとうキャンペーンを開始します - はてなブログ開発ブログ

はてなブログ1周年おめでとう！ id:hatenablog

ということは、このブログも だるだるなう - だるろぐ から約一年というわけになるのですね！　月日が経つのは早いものだ。投稿数は 494（この記事を含まず）。まぁまぁ、頑張ってる。

「はてなブログ」で気に入っているのは、画像の貼り付けが簡単で、はてな記法を利用してサクサク書けるところかな。ほんとは Markdown で書きたいけれど、個人的に足りないところが少しあって、今でもはてな記法で書いている。あと、WordPress とか管理するのがめんどくさくなった（棒。

一方、あんまり気に入らないのは、

  • 新しい記法が追加されても、記事へ反映させるために再編集が必要なこと（全記事を再生成する機能はない）
  • エクスポート機能がない（バックアップできないので、はてながデータを飛ばしたら泣くしかない）
  • はてなブログ Pro にアップグレードしたのに、はてなフォトライフ代が勝手にポイント引かれたんですけど
  • フィードバックページが梨のつぶて
  • 金をとっているのにいつまでたってもベータ版
  • その他もろもろ

当初の「はてなブログ」は「はてなダイアリー」とあまり差別化ができてなかったけれど（はてなブログの存在意義がマジで分かんない - だるろぐ）、ちょこちょこ改善されていて、最近は結構いい感じになってきたんじゃないかな。

まぁ、生暖かく見守ることにする。

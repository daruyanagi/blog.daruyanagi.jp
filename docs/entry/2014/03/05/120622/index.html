市川市民最後の日にもらったプレゼント

公開日： 2014/03/05

親父をクルマで市川駅へ送る途中に、シートベルト義務違反の違反切符をいただきました。ちょっとだけ助手席の同乗者さまがシートベルトを外すタイミングが早かった……あと30秒我慢していれば、駅についたのに。

まぁ、しょうがない。せっかくのゴールド免許だったのだけど。
